# Dashboard Feature Ideas

This document tracks potential features and improvements for the Dashboard application.

## High Priority

### Weather & Environment
- [ ] **Weather Alerts Enhancement** - Add severe weather warnings and notifications
- [ ] **Weather Radar with Animation** - Animated radar map for tracking storms
- [ ] **Air Quality Index Tracker** - Local AQI with health recommendations

### Financial & Tracking
- [ ] **YNAB API Integration** - Budget overview showing remaining amounts in categories
- [ ] **Stock Watchlist** - Track specific stocks beyond the major indices
- [ ] **Crypto Prices Dashboard** - Bitcoin, Ethereum, and other cryptocurrencies with 24hr change
- [ ] **Gas Prices Tracker** - Local gas station prices using GasBuddy API

### Family & Home
- [ ] **Family Calendar Integration** - Google Calendar showing upcoming events
- [ ] **Birthday/Anniversary Tracker** - Countdown to important family dates
- [ ] **Package Tracking Dashboard** - Track Amazon/UPS/FedEx deliveries

### News & Information
- [ ] **News Headlines Dashboard** - Top news stories (similar to sports RSS feeds)
- [ ] **Traffic/Commute Dashboard** - Real-time traffic for common routes

### Entertainment
- [ ] **TV Show Tracker** - Shows you're watching with next episode dates

## Technical Improvements

### Performance & Infrastructure
- [ ] **Photo Storage Optimization** - Move from base64 encoding to filesystem storage for better performance
- [ ] **Basic Testing Framework** - Set up Jest/Vitest for component and API testing
- [ ] **Bundle Size Monitoring** - Track and optimize JavaScript bundle sizes
- [ ] **Fix Ecosystem.config Migration** - Complete migration to ecosystem.config.js

## Completed Features

- [x] Bob's Car Wash Dashboard - Weather-based car wash recommendations
- [x] Market Overview Dashboard - Major stock indices (S&P 500, Dow, Nasdaq, Russell 2000)
- [x] Multiple Countdown Events - Support unlimited countdown timers with rotation
- [x] Countdown Navigation Controls - Manual browsing with timer reset
- [x] Error Boundaries - Crash isolation for stability on Raspberry Pi
- [x] TMDb API Key Management - Move API keys to Admin panel
- [x] Dashboard Controls Tab - Dedicated rotation settings interface
- [x] NHL Goal Summary Enhancement - Team logos in goal details
- [x] Timezone Bug Fixes - Proper ET timezone handling for games
